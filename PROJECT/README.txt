1 - Como executar?

Para iniciar um roteador basta executar no console (shell)

$ ./iniciar_router [id]

Onde [id] eh o id do roteador a ser iniciado, de acordo com o que eh setado no arquivo resources/roteador.config. Ou seja, para executar o roteador 1 executa-se:

$ ./inicia_router 1

2 - Como finalizar a execução de um roteador?

Durante a execucao de um roteador, pode ser pressionado a qualquer momento as teclas Ctrl + C e finalizar-se o processo.
